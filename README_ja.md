## 概要

気候変動は、一般に温室効果ガス、とくに二酸化炭素（CO₂）の増加によって説明される。

この説明は、大気物理のレベルでは科学的に正しい。  
CO₂は温室効果ガスであり、その蓄積は放射強制力を高め、地球温暖化に寄与する。

しかし、説明が「排出」だけで止まるなら、それは不完全である。

本稿が示すより深い因果層は、地球の自然な炭素固定、炭素吸収、水循環、土壌、森林、海洋、湿地、微生物循環システムの劣化である。

つまり、大気中CO₂が増えたのは、人間文明がCO₂を排出したからだけではない。  
地球が炭素を吸収し、固定し、貯蔵し、再循環する自然能力そのものが弱体化してきたからでもある。

従来の単純化された因果モデルは、次のように表される。

```text
人間活動
→ 温室効果ガス排出の増加
→ 地球温暖化
→ 気候変動
```

本稿では、より深い因果モデルを提案する。

```text
人間活動
→ 土壌・森林・海洋・湿地・微生物循環の破壊
→ 炭素固定源と炭素吸収システムの崩壊
→ CO₂が吸収・固定・再循環されにくくなる
→ 大気中CO₂が蓄積する
→ 地球温暖化が加速する
→ 水循環・生態系・海洋・農業・気象システムが不安定化する
→ 気候変動が激化する
```

IPCCは、人間の影響が大気、海洋、陸域を温暖化させてきたことは疑う余地がないと述べている。  
本稿はその結論を否定しない。

本稿が主張するのは、公共の気候コミュニケーションでは、その前段階にある生態学的原因、つまり炭素循環を制御する自然システムの破壊が十分に説明されていない、という点である。

## 2. なぜ従来の説明は不完全なのか

気候変動の主流説明は、一般に次のように単純化される。


これは間違いではない。

しかし、不完全である。

それはプロセスの後半を説明しているが、完全な因果連鎖を説明していない。

欠けている問いは、次である。

```text

```

答えは、単に以下ではない。

```text

```

より深い答えは、以下である。

```text

```

もし土壌、森林、海洋、湿地、微生物、生物循環システムがより健康な状態を保っていれば、大気中の炭素のより大きな部分は吸収、固定、貯蔵、再循環されていた可能性がある。

これは、CO₂排出が無害だという意味ではない。

CO₂蓄積は、原因であると同時に結果でもある、という意味である。

CO₂は大気レベルでは温暖化の原因である。  
同時に、CO₂蓄積は惑星システムレベルでは炭素循環不全の結果でもある。

目的は単に排出を減らすことではない。

地球の炭素代謝と自然循環システムを回復することである。

## 16. 結論

気候変動は、CO₂排出だけによって起きているわけではない。

CO₂は地球温暖化の直接的な物理要因である。  
しかし、その継続的蓄積は、より深い惑星的失敗の症状でもある。

その失敗とは、炭素固定、炭素吸収、微生物循環、自然調整システムの崩壊である。

地球が不安定化しているのは、人間が炭素を排出しているからだけではない。

人間文明が、炭素、水、栄養、生命を吸収し、固定し、貯蔵し、冷却し、循環させ、再利用していた自然システムを傷つけてきたからである。

したがって、次の段階の気候対策は以下である。

```text
排出削減
+
炭素固定源回復
+
微生物・生態循環回復
+
水循環・生態系回復
```

気候戦略の未来は、以下へ移行すべきである。

```text
脱炭素だけから
惑星炭素代謝の回復へ。
```

あるいは、より簡潔に言えば、

```text
CO₂を減らすことから
地球がCO₂を処理する能力を回復することへ。
```
